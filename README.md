# Presidential Panic

A simple sidescrolling shoot-em-up developed as a school project.
Developed using Unity 5.4.

## Controls

 - WASD to move.
 - Space to jump.
 - K to shoot.
 - If you have a machine gun, hold K to go full-auto.
 - Escape to access menu.
   - K to accept.
   - Space to back.
   - WS to move up and down.
